# Continuer

여러 환경에서(PC, 모바일) 자유롭게 사용할 수 있는 타임라인형 저널 애플리케이션

👉 [사이트](https://continuer-93336.web.app/)

## 기간, 인원

- 2021.03.17 ~ 4.19 (1개월)
- 개인 프로젝트

## 관련 기술

- Typescript
- ReactJS
- Redux, Redux Toolkit
- styled-components
- firebase

## 구현 사항

- mobile first 반응형 웹
- PWA
- 역방향 인피니티 스크롤
- Intersection Observer를 이용한 Header 컴포넌트 날짜 변경
- 이미지 파일 업로드
